# Configuración JavaFX

Para ejecutar el proyecto se debe configurar el apartado **Run VM Options**.

## Configuración utilizada en casa:
--module-path "C:\openjfx-21.0.11_windows-x64_bin-sdk\javafx-sdk-21.0.11\lib" --add-modules javafx.controls,javafx.fxml

## Configuración utilizada en el colegio:
--module-path "C:\javafx-sdk-21.0.11\lib" --add-modules javafx.controls,javafx.fxml


La configuración depende del equipo donde se ejecute el proyecto.
